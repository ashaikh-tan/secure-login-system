# Secure Login System

A secure login and registration web application built using Flask and SQLite.

## Features

- User registration and login
- Password hashing using bcrypt
- Session management
- Logout functionality
- SQL injection protection
- Secure authentication flow

## Technologies Used

- Python
- Flask
- SQLite
- bcrypt

## Security Features

- Hashed passwords
- Parameterized SQL queries
- Session-based authentication
